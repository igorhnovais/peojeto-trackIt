## Requisitos do projeto

Esse projeto teve o intuito de fazer um site mobile de controle de hábitos diários com Api no prazo de 5 dias, com certos requisitos infra-listados:
 - se o usuario não tiver conta, terá que cadastrar uma.
 - se ele já tiver conta, entrar com o seu login e senha.
 - entrando no site, ele verá os hábitos do dia.
 - se tiver feito o hábito, ele marcará com check o habito realizado, e poderá desmarcar esse hábito também.
 - se ele quiser criar um novo hábito, ele mudará para a pagina de hábitos e poderá criar um novo hábito ou excluir um já existente.
 - e se ele quiser, poderá ir no seu histórico, porém será uma pagina estática.
